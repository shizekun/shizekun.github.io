Vue基础

1.Vue介绍

概念：Vue (读音 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与现代化的工具链以及各种支持类库结合使用时，Vue 也完全能够为复杂的单页应用提供驱动。

  • 版本：
    • 主流： 2.6
    • 最新：3
  • 作者：尤雨溪
  • 框架： JS框架！！！
    • UI框架： bootstrap!

  • 特点：易用，灵活，高效

  • 兼容：IE8以下不兼容

使用：

1.引入vue.js

开发版本(development)

生产版本(production)

2、创建挂载节点标签

不能是body或者html

3、实例化vue对象

el:挂载节点的选择器

data:模板中用到的所有数据(变量，表达式)

MVVM:

M model（data里面的数据）

V view(视图，就是模板)

VM (虚拟模型)

注意：data里面的数据一旦变化，模板就会自动更新

2.Vue指令

指令是什么： 是自定义属性 <标签 v-指令名=”变量/表达式” />

常用指令：

v-html 渲染内容，可以编译html字符串

v-text 渲染内容

v-cloak 防止闪烁

v-pre 跳过编译

v-once 一次性绑定

v-on 事件绑定

v-bind 属性绑定

v-model 双向绑定

v-if

v-else-if

v-else

v-for

3.Vue事件

绑定：

<标签 v-on:事件类型=”事件函数” />

<标签 v-on:事件类型=”事件函数()” />

<标签 @事件类型=”事件函数” />

传参：

<标签 @事件类型=”事件函数(实参)” />

事件对象：

<标签 @事件类型=”事件函数” />

<标签 @事件类型=”事件函数($event)” />

既传参又获取事件对象:

<标签 @事件类型=”事件函数($event,实参)” />

事件简写：

<标签 @事件类型=”变量赋值操作” />

事件修饰符:
修饰符:

<标签 @事件类型.修饰符=”事件函数” />

修饰符内容：

.stop .prevent .self .capture .once

按键修饰符:

keydown keyup keypress input

注意：只能在 input 和textatea

自定义按键修饰符:

Vue.config.keyCodes.修饰符名 = 按键码

4.Vue样式

class:

对象：<标签 :class=”{ class名1:变量1，class名2:变量2 }” />
数组：<标签 :class=”[变量1，变量2,’class名1’,’class名2’]” />
属性绑定：<标签 :class=”条件 ？ ‘class名1’:’class名2’ “ />

style:

对象：<标签 :style=”{ css属性名名1:变量1，css属性名2:变量2 }” />
数组：<标签 :style=”[变量1，变量2]” />

变量1:{ css属性名:css属性值 }

变量2:{ css属性名:css属性值 }

属性绑定：
<标签 style=”font-size:20px;border-color:red” />
<标签 :style=”font-size:${a};border-color:${b}“ />
